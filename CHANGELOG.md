## Version 2.9.0

### New Features
- Added folder context menu for quick batch upload to Notion
- Right-click on any folder to see "Sync to [Database]" options
- Nested folder structure support confirmed and working

### Changes
- Default tags functionality is now disabled by default (can be enabled in settings)
- 添加文件夹右键菜单快速批量上传功能
- 右键点击文件夹即可看到"同步到 [数据库]"选项
- 确认并优化嵌套文件夹结构支持
- 默认关闭 tags 功能(可在设置中启用)

---

## Previous Features

- Better debugging information for upload failures.
- Support synchronising Obsidian callouts as Notion callout blocks.
- 优化控制台输出的调试信息。
- 支持将 Obsidian Callout 同步为 Notion Callout 区块。
